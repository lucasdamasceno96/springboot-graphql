# 🚧 Como fazer funcionar

```bash
# Clone o Repositorio
$ gh repo clone https://github.com/lucasdamasceno96/springboot-graphql.git
$ Abra a pasta no VsCode
$ Baixe as dependencias 
$ Execute o projeto
$ http://localhost:8080/graphiql?path=/graphql

```
